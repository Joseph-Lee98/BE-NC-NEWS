# Northcoders News API



This backend application makes use of two databases: One database for development, and one for testing. In the root directory, you will need to create two .env. files in order to create the environment variables necessary to connect to the required database. 

Create a file named '.env.development', and in this file write the following code: PGDATABASE=nc_news

Also create a file named '.env.test', and in this file write the following code: PGDATABASE=nc_news_test
